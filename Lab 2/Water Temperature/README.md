# Water Temperature Analysis

Analysis of water temperature data from the CalCOFI (California Cooperative Oceanic Fisheries Investigations) dataset.

## Dataset

**Source:** [CalCOFI - California Cooperative Oceanic Fisheries Investigations](https://www.kaggle.com/datasets/sohier/calcofi)

### About the Dataset
The CalCOFI dataset contains oceanographic data collected by the California Cooperative Oceanic Fisheries Investigations program. It includes measurements of temperature, salinity, oxygen, and other parameters from water samples collected off the coast of California.

## Files

- `temperature.ipynb` - Main analysis notebook for water temperature data

## Setup Instructions

1. Download the dataset from Kaggle:
   - Visit: https://www.kaggle.com/datasets/sohier/calcofi
   - Click "Download" button
   - Extract the CSV files to this folder

2. Install required dependencies:
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn jupyter
   ```

3. Open and run the notebook:
   ```bash
   jupyter notebook temperature.ipynb
   ```
